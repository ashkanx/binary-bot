Welcome to the Binary Bot!

Binary Bot is a web application designed for non-programmer (or even programmer) users to program trading robbots. Your programmed blocks will generate codes that work directly with the binary.com API. You can pick many different blocks to program simple or complex robbots.

A basic background of programming is required to work with Blockly and therefore Binary Bot. You can find good material to get started and build your computational thinking skills through these tutorials: Computational Thinking Introduction to Blockly:

  • Introduction to Blockly

Binary Bot is a visual programming environment based on Google Blockly.
